# Sistema ABM para la ONG Banco Alimentario de La Plata

## Introducción
El BALP o Banco Alimentario de La Plata asiste a diferentes comedores, hogares y merenderos de la zona de La Plata, Berisso y Ensenada. Se dedica al recupero de alimentos que por diversas razones no pueden ser comercializados pero son aptos para el consumo humano. Hoy en día existen más de 140 instituciones donde asisten niños, adultos y ancianos que reciben la ayuda del BALP. 
http://bancoalimentario.org.ar/
http://www.linkedin.com/company/banco-alimentario-de-la-plata

## Objetivos de este sistema ABM (alta, baja, modificación)
Este sistema será de uso interno dentro de la ONG. Lo usarán los voluntarios y empleados del Area Social, que se encargan de atender las necesidades de cada institución y realizar visitas de seguimiento y actualización de datos. Para ello necesitan un sistema que sea fácil de usar y que permita consultar información sobre cada institución fácilmente. Actualmente dependen para las consultas de un archivo excel y de las carpetas físicas con la documentación en papel.
La información disponible de cada institución está organizada en formato tabla y se guarda dentro de una base de datos. El sistema permite además modificar algún dato existente y eliminar instituciones que sean dadas de baja de la ONG (debido a que por ejemplo dejan de funcionar, no brindan su servicio adecuadamente o venden la mercadería recibida).
Se requiere una autenticación para entrar al sistema. Esto evita que personas ajenas a la ONG puedan acceder a estos datos.

## Herramientas utilizadas
Javascript ES6, Express, NodeJS, Axios, MongoDB, React.
